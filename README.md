# Disease Prediction from Medical Data

## Project Overview
This project predicts whether a tumor is benign or malignant using the Breast Cancer Wisconsin dataset and a Random Forest classifier.

## Technologies Used
- Python
- Pandas
- Scikit-learn

## Dataset
Breast Cancer Wisconsin Dataset (Built into Scikit-learn)

## Features
- Data preprocessing
- Random Forest Classification
- Model evaluation
- Disease prediction
- Accuracy, Precision, Recall and F1-Score

## Result
The model achieves around 95–99% accuracy on the test dataset.
